# DrumMidiEditorApp

[to Wiki Page](../../wiki)

本アプリは、ドラムの耳コピ用に作成したMIDIエディターのようなアプリです。  

特徴  
・ドラムの打込みに特化した構成  
・BGMを再生しながらのNOTE打込み  
・NOTE打込み情報を別ウィンドウでシーケンス確認が可能
